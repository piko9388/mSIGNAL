누적관리
매주 발행되는 Weekly Report를 정형 JSON으로 누적 저장하고, 시간이 흐를수록 의미 있는 트렌드·순위·예측 데이터로 진화시키는 영역입니다. Weekly가 5건 이상 누적되는 시점부터 본격 활성화됩니다.

01활용 시나리오

📊 누적 데이터로 가능해지는 분석

①
BP 언급 빈도 누적 랭킹 — 어떤 BP가 시장에서 가장 활발히 움직이는가. 분기별·반기별 Top 20 자동 산출
②
소재군별 Signal 강도 추이 — Photo·Chemical·CMP·Gas·Precursor·Wafer·Target별 시계열. 특정 소재군 급증 시 공급망 위기 조기 감지
③
신뢰도 분포 변화 — ● 공식 / ◐ 업계 / ○ 추정 비중 추이. ○ 비중 상승 = 정보 부정확·확인 부족 시그널
④
키워드 동시발생 네트워크 — co-occurrence 매트릭스. HBM4E ↔ 1c나노 ↔ 후공정 같은 클러스터 자동 추출
⑤
Key Signal 회수율 추적 — 주차별 임팩트 강도 비교. 시장이 조용했던 주 vs 격동의 주 정량 구분
⑥
BP × 카테고리 히트맵 — pivot table 자동 생성. 특정 BP의 카테고리 편중도 확인
⑦
계절성 패턴 — 분기별 Signal 분포. 실적 발표 시즌·CES·SEMICON 영향도 정량화
